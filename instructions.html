INSTRUCTIONS

Overview

Footy Timer is intended to help footy (AFL) timekeepers keep track of when to play sirens.

Quarter Selection

  • Tap Q1-4 to highlight it.
  • Tap again to deselect it.
  • This helps keep track of the current quarter.

Game Time

This is the main timer.

  • Choose the length of the quarter in minutes from the drop down menu.
  • Choose whether the timer counts up or down.
  • Tap set to enter the time and direction.
    • NOTE: Selections only become active when set is pressed.
    NOTE: Selections don't update in pause mode.
  • Tap run to begin the timer.
  • When the the timer reaches zero the siren will sound.
  • The intermission timer will AUTOMATICALLY begin.
    • Tip: make sure the intermission timer has been set!

Intermissions

  • Set the duration of the intermission (in minutes) using the drop down menu.
  • Tap set to update the intermission timer.
  • Tap run to start it counting down.
    • NOTE: The intermission timer will AUTOMATICALLY start when the game timer reaches zero.
  • The next quarter will NOT automatically start.

Sirens

Sirens can be operated manually using the three buttons in this section.

  • It is recommended you stop the timers if you are operating them manually.
    • Tip: This may be useful if you are connected to a speaker.

Notes & Contact
